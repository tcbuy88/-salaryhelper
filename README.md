# SalaryHelper - Delivery Package

This package contains a static front-end demo (H5 + Admin), a small FastAPI server stub, OpenAPI mock spec and supporting docs for SalaryHelper (劳动纠纷助手).

Usage:
1. Inspect files in salaryhelper-delivery/
2. Start mock + static + stub:
   docker compose -f docker-compose-full.yml up --build
3. Or run server stub:
   cd server && pip install -r requirements.txt && uvicorn app.main:app --reload --port 8000
